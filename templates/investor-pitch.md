# Investor Pitch Template

A 15-slide structure for investor meetings and fundraising presentations.

## Template Structure

Use this structure when creating investor pitches. Each section becomes a slide.

```
[Title]
[Company Name] - Investor Presentation
[Date] | [Your Name]
---
[Problem]
The problem we're solving
- Current pain points
- Why existing solutions fail
- Market gap
---
[Solution]
Our approach to solving this problem
- Core product/service
- Key differentiators
- Why now
---
[Product]
What we've built
- Core features
- How it works
- User experience
---
[Demo/Screenshots]
See it in action
- Key workflows
- Interface highlights
- Real examples
---
[Market Opportunity]
The size of the opportunity
- TAM/SAM/SOM
- Growth trends
- Market dynamics
---
[Business Model]
How we make money
- Revenue streams
- Pricing model
- Unit economics
---
[Traction]
What we've achieved
- Key metrics
- Growth trajectory
- Milestones
---
[Competition]
The competitive landscape
- Key players
- Our differentiation
- Moat/defensibility
---
[Go-to-Market]
How we reach customers
- Acquisition channels
- Sales strategy
- Growth levers
---
[Team]
The people behind the company
- Founders and leadership
- Key hires
- Advisors
---
[Financials]
Current state and projections
- Revenue (actual/projected)
- Key metrics
- Path to profitability
---
[The Ask]
What we're raising
- Round size
- Use of funds
- Timeline
---
[Vision]
Where we're going
- Long-term vision
- Roadmap highlights
- Impact
---
[Contact]
Let's continue the conversation
- Contact information
- Next steps
- Q&A
```

## Usage Notes

### Tone
- Confident but not arrogant
- Data-driven claims
- Clear and concise

### Customization Points
- Replace [placeholders] with actual content
- Add or remove slides as needed
- Emphasize sections relevant to your story

### Common Variations
- **Seed round**: More focus on vision, team, problem
- **Series A**: More focus on traction, metrics, GTM
- **Series B+**: More focus on financials, expansion

## Gamma Parameters

When using this template:
```json
{
  "textMode": "generate",
  "numCards": 15,
  "textOptions": {
    "tone": "professional, confident, compelling",
    "audience": "investors, VCs, angels"
  }
}
```
